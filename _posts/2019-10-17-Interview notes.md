```javascript
function test() {
  console.log(a);
  console.log(foo());
  var a = 1;

  function foo() {
    return 2;
  }
}
test();

// 結果會印出
// undefined
// 2

// 記得在creation phase時:
// setup memory space for variables and functions!!
// 執行到console.log(a)時 a只被宣告 還沒被賦值 所以印出undefined
// 而foo()函數已經被放入記憶體中! 所以印出2
```

還有經典的let v.s. var

```javascript
 for (var  i = 1; i < 5; i++) {
  setTimeout(function() {
    console.log(i);
  }, 1000);
} 
// 結果會印出4次5
// var 會將變數宣告為global
// 當執行到setTimeout時 i為5

```
